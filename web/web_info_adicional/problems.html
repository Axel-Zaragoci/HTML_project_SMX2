MENÚS DESPLEGABLES

Para los menús desplegables de la cabecera, no tenía idea de como hacerlo. Y eso era un problema porque quería tener menús desplegables en la cabecera para que el nav no quedase demasiado largo. Debido a esto, empecé a buscar hasta encontrar una página que me aprendió a hacerlo. Más adelante, mientras hacía la página de formulario se me ocurrió que simplemente podría haberlo hecho con un select

CARRUSEL DE IMÁGENES

Para el carrusel de imágenes me pasó algo parecido. Yo quería tener al menos un carrusel de imágenes automático en mi sitio web, pero no tenía ni una pista de como hacerlo. Debido a eso empecé a buscar, pero solo encontraba tutoriales que usaban JavaScript, pero tras mucho buscar acabé encontrando un tutorial en youtube que lo hacía solo con HTML y CSS, por lo que lo implementé y tuve que cambiar algunos valores de anchura para que se adaptase a mi página

EFECTO PLACEHOLDER-LABEL

Para las páginas de inicio de sesión y de registro quería hacer un efecto con los placeholders y los labels, tal y como ya hice en un curso en el que participé. Me costó mucho el lograr repetir el efecto y hasta tuve que revisar mis antiguos códigos para ayudarme con algunos pequeños detalles, como por ejemplo mi duda de ocultar los label usando display: none; o visibility: hidden; opacity: 0;

VALIDACIÓN EN EL FORMULARIO

A la hora de crear la validación de los formularios para las contraseñas, no lograba hacerlo funcionar. Tras una larga e intensiva búsqueda por internet, tampoco encontré nada que resolviese el problema usando solo HTML y CSS. Al final, acabé recoriendo a una IA que me enseñó mi novia para programar, blackbox, y con esta logré crear y retocar el código para que funcionase usando exclusivamente HTML y CSS
